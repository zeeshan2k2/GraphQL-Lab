# GraphQL Lab

A learning-focused Swift repository exploring GraphQL integration in iOS applications. The project demonstrates how GraphQL queries, mutations, and schema-driven data fetching can be used to efficiently communicate with backend services.

---

## Overview

This repository contains practical examples built while learning GraphQL in iOS. The goal is to understand how GraphQL differs from traditional REST APIs, how queries and mutations are structured, and how strongly typed responses can be integrated into Swift applications.
